## Display Font

We use Cantora One as display font. It has only one weight und should be used for very large text only (`xl`, `2xl`, `3xl` and `4xl`.

<Typeset
  fontFamily="Cantora One"
  fontSizes={["1.5625rem", "2rem", "2.5rem", "3rem"]}
/>

## Body Font

Laila is our standard body font with five different weights. Possible size modifiers are `xs`, `sm`, `base`, `lg`, `xl` and `2xl` though usually sticking with the base size and the `sm` variant should be sufficient.
